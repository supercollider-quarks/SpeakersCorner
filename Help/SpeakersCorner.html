SpeakersCorner


A set of classes for Multi-Speaker setups.


SpatioScope - display speaker layout in 2D, show audio levels on the speakers


Latency - measure audio latencies for each channel, to compensate distances 

  to a given listening/microphone location


Spectralyzer - display frequency response of a signal, e.g. a loudspeaker by microphone


SpeakerAdjust - adjust delaytime, level and EQ for individual speaker channels


MasterFX - add switchable master effects on the main outputs, 

  e.g. using SpeakerAdjust, limiters, etc.



Panning Classes:


TorusPanAz for concentric rings of speakers

PanRingTop for ring + a top speaker

PanRingTopBot for a ring, top and bottom speaker
